# 溧阳制造企业 AI 获客平台 Demo

本项目不是单个企业官网，而是面向溧阳制造企业的可复制 AI 获客平台 Demo。

这是一个用于 TRAE AI 创造力大赛演示的静态网站项目，面向溧阳本地小工厂、加工厂和制造企业。

## 项目定位

帮助制造企业快速生成：

- 企业官网首页（双角色入口：工厂老板 / 采购客户）
- 企业展示页
- 产品展示页
- 招聘展示页
- 联系我们页面
- AI 报价助手（含导出报价单）
- 管理后台（AI 一键生成企业介绍）
- 询盘管理（客户询盘跟踪）
- 参赛说明页

示例企业为"溧阳五四不锈钢"，主营不锈钢零部件、CNC 加工和数控车加工，服务新能源行业。

## 文件结构

```text
liyang-ai-manufacturing-platform/
├── index.html          # 首页：双角色入口 + 平台价值 + 询盘流程 + 溧阳制造业适配
├── enterprise.html     # 企业展示页：溧阳五四不锈钢
├── products.html       # 产品展示页：不锈钢零部件、CNC 加工、数控车加工
├── quote.html          # AI 报价助手：完整报价分析 + 微信话术 + 导出报价单

├── admin.html          # 管理后台：企业信息录入 + AI 一键生成企业介绍
├── jobs.html           # 招聘展示页：CNC 操作工、数控车工、质检员
├── contact.html        # 联系我们页面

├── css/

└── js/
    └── app.js          # 导航交互 + AI 报价助手 + 管理后台 + 询盘管理
```

## 运行方法

方法一：直接打开

1. 进入项目文件夹。
2. 双击 `index.html`。
3. 使用浏览器查看网站。

方法二：使用本地静态服务

如果电脑已安装 Python，可在项目目录执行：

```bash
python3 -m http.server 8080
```

然后在浏览器打开：

```text
http://localhost:8080
```

## 评委演示步骤

第 1 步：打开首页，先查看顶部“评委演示路线”，理解完整体验路径。

第 2 步：进入管理后台填写企业信息，包括企业名称、主营产品、设备数量、服务行业、企业优势、电话、微信和地址。

第 3 步：点击“一键生成企业介绍”，查看 AI 自动生成的企业简介、产品介绍、招聘文案和客户沟通话术；也可以点击“保存企业信息”，刷新页面后验证自动回显。

第 4 步：进入 AI 报价助手，点击“一键填充示例需求”，再生成完整报价分析。

第 5 步：点击“导出报价单（打印）”，查看可打印的客户报价单页面。

第 6 步：进入询盘管理，查看客户询盘列表，并点击“更新状态”体验待联系、已报价、已成交的跟进流程。

第 7 步：查看参赛说明，理解项目不是单独企业官网，而是可复制给溧阳上百家制造企业使用的 AI 获客工具。

## 演示重点

- **首页**：双角色入口（工厂老板 → 管理后台 / 采购客户 → AI 报价），突出"让每一家溧阳制造企业，都拥有自己的 AI 官网和获客助手"的大标题，展示平台价值数据、客户询盘流程和溧阳制造业适配优势。
- **管理后台（admin.html）**：录入企业名称、主营产品、设备数量、服务行业、企业优势、联系方式后，AI 一键生成企业简介、产品介绍、招聘文案、客户沟通话术和联系方式汇总。
- **询盘管理（leads.html）**：展示客户询盘列表，包含客户姓名、需求产品、数量、工艺、AI 报价结果和跟进状态（待联系 / 已报价 / 已成交），支持点击更新状态，顶部统计面板实时同步。
- **AI 报价助手（quote.html）**：可根据材料、数量、工艺、交期和补充说明生成完整报价分析，包括报价区间、加工难度、交期建议、风险提醒和可直接复制给客户的微信回复话术，支持一键导出可打印的报价单。
- **企业展示页**：展示设备能力、行业方向和企业优势。
- **产品展示页**：展示不锈钢零部件、CNC 加工、数控车加工。
- **招聘展示页**：展示 CNC 操作工、数控车工、质检员岗位。
- **联系我们页面**：预留电话、微信、地址占位信息。
- **参赛说明页**：详细介绍项目解决的问题、目标用户、AI 能力、Demo 功能、商业模式和未来推广计划。

## 技术说明

- 纯 HTML + CSS + JavaScript。
- 不依赖后端。
- 不依赖第三方框架。
- 所有页面可直接静态部署。
- 支持响应式设计，适配移动端和桌面端。
- 导航栏统一：首页、企业展示、产品展示、AI 报价、询盘管理、管理后台、招聘、联系我们、参赛说明。
